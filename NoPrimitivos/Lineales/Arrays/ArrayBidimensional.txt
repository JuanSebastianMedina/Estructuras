Un array bidimensional es una estructura 
de datos lineal compuesta por arrays. Se 
utiliza para representar tablas o matrices.
Cada fila es un array independiente.
Por eso Java permite arrays irregulares.

ejemplo inicializacion directa:
int[][] matriz = {
 {1,2,3},
 {4,5,6},
 {7,8,9}
};

declaracion:
tipo[][] nombre;

creacion:
int[][] matriz = new int[3][4];

acceso individual:
int valor = matriz[1][2];

acceso completo:
for(int i = 0; i < matriz.length; i++){
    for(int j = 0; j < matriz[i].length; j++){
        System.out.print(matriz[i][j] + " ");
    }
    System.out.println();
}
